# Visual_Vector

### 这是一个小项目

本项目为无锡市第一中学高一（1）班某不知名研学小组的研学课题，目标是制作一个可以直观显示并计算平面向量的、~~抽象~~好用的小软件。

目前本项目处于刚起步阶段，计划使用Python语言编写GUI，C++语言编写向量类（主要是出于分工原因）。
